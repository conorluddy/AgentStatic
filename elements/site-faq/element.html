Frequently Asked Questions
Why no JavaScript?

Most landing pages don't need JavaScript. Removing it makes sites faster, more accessible, and easier to host.

How do custom elements work without registration?

Browsers render unknown elements as inline by default. We use CSS to style them as block, grid, or flex. No JavaScript needed.

Can I add JavaScript for interactivity?

Absolutely! The build system doesn't prevent JavaScript. Add it when you need progressive enhancement.

What about dark mode?

Dark mode is built-in using CSS custom properties and the prefers-color-scheme media query.
