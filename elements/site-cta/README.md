# site-cta

Call-to-action section for driving conversions.

## Usage

```html
<site-cta>
  <h2>Ready to get started?</h2>
  <p>Join thousands of happy customers.</p>
  <site-cta-actions>
    <a href="/signup" class="button">Start Free Trial</a>
  </site-cta-actions>
</site-cta>
```

## Variants

| Attribute | Values | Description |
|-----------|--------|-------------|
| `theme` | `dark` | Dark background variant |

## Child Elements

- `h2` — Headline
- `p` — Supporting text
- `site-cta-actions` — Button container
