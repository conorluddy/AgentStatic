Frequently Asked Questions
Why no JavaScript?

Client-side JavaScript adds complexity, slows down page loads, and can break. For content-focused sites like landing pages, pure HTML and CSS delivers a faster, more reliable experience for your visitors.

How do I change the brand colors?

Edit brand.json with your primary color (e.g., "#3b82f6"), then run npm run build. The system generates a full color palette including shades from 50 to 950, plus semantic colors for backgrounds, text, and accents.

Can I add my own components?

Yes! Create a folder in elements/ with your element name (e.g., site-testimonials), add a styles.css file, and write your HTML. The build system automatically includes your new component styles.

How do I deploy the site?

Run npm run build to generate the dist/ folder, then deploy it anywhereâ€”Netlify, Vercel, GitHub Pages, or any web server. It's just HTML and CSS files.

Is this good for SEO?

Excellent. Pure HTML means search engines see your content immediately without waiting for JavaScript to render. Fast load times also improve your Core Web Vitals scores.
