## Getting Started

To customize this blog:

1. Edit or delete the sample posts in `pages/blog/`
2. Create new `.md` files with frontmatter (title, description, date)
3. Run `npm run build` to regenerate the site

## Frontmatter

Every blog post needs frontmatter at the top:

```yaml
---
title: Your Post Title
description: A short description for listings and SEO
date: 2025-01-15
---
```
