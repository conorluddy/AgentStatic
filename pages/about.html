About AgentStatic

A static site generator built for simplicity.

Why We Built This

Modern web development has gotten complicated. Build tools with hundreds of dependencies, hydration bugs, slow page loads, and framework churn. For many sites—especially landing pages and marketing content—all that complexity isn't adding value.

AgentStatic takes a different approach. Your output is pure HTML and CSS. No JavaScript frameworks, no client-side rendering, no bundle sizes to worry about. Pages load instantly because there's nothing to parse or execute.

Who It's For

AgentStatic is designed for people who want to build content-focused websites without framework overhead:

  • Technical marketers who can edit HTML and want fast results
  • Developers who want a simpler stack for landing pages
  • Agencies building lots of client sites quickly
  • Anyone tired of JavaScript build tool complexity

How It Works

The entire build system is around 330 lines of TypeScript across four files. It does three things:

  1. Generates CSS tokens from your brand.json config
  2. Concatenates all CSS into a single stylesheet
  3. Expands include-* elements in your HTML pages

That's it. No bundlers, no tree-shaking, no code-splitting decisions. Just simple transformations you can understand and debug.

The Philosophy

  • Simplicity over features — Do less, do it well
  • HTML is fine — You don't need JSX for landing pages
  • CSS is powerful — Custom elements + CSS selectors work great
  • Fast by default — No JavaScript means no performance problems
