Build static sites the AgentStatic way

Getting Started

AgentStatic is a static site generator that compiles TypeScript to produce pure HTML and CSS. No JavaScript in your output by default.

Project Structure

├── brand.json           # Site config
├── pages/               # Your pages
├── elements/            # Component library
├── partials/            # Reusable fragments
├── build/               # Build scripts
└── dist/                # Output (generated)

Quick Commands

npm run build    # Build once
npm run dev      # Build + watch
npm run serve    # Serve locally

Brand Configuration

Control your entire site's look and feel through brand.json:

{
  "name": "Your Site",
  "tagline": "Your tagline",
  "colors": { "primary": "#22c55e" },
  "typography": {
    "fonts": { "sans": "Inter", "mono": "JetBrains Mono" }
  }
}

What It Generates

  • Color tokens: --color-primary-50 through --color-primary-950
  • Spacing scale: --space-1 through --space-32
  • Typography: --text-xs through --text-5xl
  • Font loading: Automatic Google Fonts integration

Creating Pages

Pages live in pages/ and use built-in components:

<!DOCTYPE html>
<html lang="en">

<include-head
  title="Page Title"
  description="Meta description"
/>

<body>
  <include-nav logo="/assets/logo.svg" />

  <site-hero>
    <h1>Welcome</h1>
    <p>Your content</p>
  </site-hero>

  <include-footer siteName="Your Site" />
</body>

</html>

Element Naming

Prefix Purpose Examples
include-* Build-time partials include-head, include-nav
site-* Page sections site-hero, site-features
ui-* Small UI bits ui-button, ui-badge

Using Partials

Partials are reusable HTML fragments with slots for dynamic content:

<include-footer siteName="My Site">
  <nav slot="links">
    <a href="/about">About</a>
    <a href="/contact">Contact</a>
  </nav>
</include-footer>

How Includes Work

  • <include-{name}> loads partials/{name}.html
  • Attributes are passed as variables: {{attributeName}}
  • <slot> receives default children
  • <slot name="x"> receives slot="x" children
