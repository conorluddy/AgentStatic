# AgentStatic: Quick Start Guide

**For**: Getting up to speed on AgentStatic planning in 5 minutes

---

## 30-Second Summary

Building a complete, standalone static site generator for brochureware in 7 phases over 12-14 weeks:

- **Template-first**: Clone and start with beautiful example pages
- **20-30 components**: Focused library with extensible framework
- **Claude-optimized**: MCP tools designed for Claude specifically
- **Predefined theme**: Beautiful defaults, easy customization
- **Zero runtime JS**: Pure CSS + semantic HTML

**Target**: Production-ready SSG, Claude can autonomously build pages from briefs.
