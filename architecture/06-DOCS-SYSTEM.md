# 06 – Documentation System

## Purpose
Provide comprehensive documentation for both humans and AI agents.

## Layers
1. Storybook Visual Browser
2. Developer Guide
3. AI Agent Guide
4. Contribution Guide
5. In-Code Documentation

## Storybook Setup
- Configured with Vite/Web Components
- a11y, viewport, dark-mode addons
- Stories auto-generated from registry

## Acceptance Criteria
- 50–70 stories visible in Storybook
- Dark/light modes functional
- a11y audit integrated
